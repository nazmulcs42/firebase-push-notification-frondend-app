## Getting Started
First, config the development server:

```bash
npm install
```
Second, config the environment file:

```bash

cp .env.example .env
```

Finally, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
